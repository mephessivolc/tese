\chapter{Proposta} \label{cap:proposta}

\modificar O algoritmo proposto está dividido em 3 etapas interelacionadas: 
\begin{itemize}
    \item \modificar\textbf{Primeira etapa} se propõe a verificar se a capacidade de modelagem e consistencia dos computadores quânticos fotônicos é mantido para solucionar problemas combinatórios. Para isso foi proposto que os qmodes fossem relacionados de 1 para 1 a qubits. Resultados obtidos mostraram que apesar do esforço computacional em modelar qmodes os resltados se mantiveram coerentes quando comparados entre si e um modelo clássico de força bruta. 
    
    \item \modificar\textbf{Segunda etapa} A modelagem matemática foi alterada para que cada qmode fosse relacionado a posição do vértice em um passo de tempo buscando o caminho, ou seja, um estado formado por 4 qmodes, como por exemplo: $[3,2,1,4]$ significa que o primeiro vértice será o terceiro a ser cruzado no caminho, o segundo vértice será o segundo a ser cruzado no caminho, o terceiro no primeiro e o quarto vértice será o último, com isso, onde cada cidade é representada por um vértice teremos o caminho: $V_0 \rightarrow V_3 \rightarrow V_2 \rightarrow V_1 \rightarrow V_4 \rightarrow V_0$. Com os resultados obtidos foram comparados os resultados através desse modelo com os resultados de força bruta, mostrando que é possível encontrar o menor caminho entre os caminhos possíveis.

    \item \modificar\textbf{Terceira etapa} A ultima etapa será a construção de um modelo de otimização para encontrar um menor caminho dentro de uma quantidade $N$ de cidades (vértices).

\end{itemize}
